## ğŸš€ Installation & Setup

### Prerequisites

- [Hugo](https://gohugo.io/getting-started/installing/) (version 0.125.1 or later)
- [Node.js](https://nodejs.org/) (version 20.11.0 or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/sujith-eag/personal-site.git
   cd personal-site
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
   This will start Hugo server with live reload at `http://localhost:1313`.

### Building for Production

```bash
npm run build
```

This generates the static site in the `public/` directory.

### Preview Build Locally

```bash
npm run preview
```
